### Setup

* Clone the repo
* Navigate to the directory and create a python 3 virtual environment and activate it
* Install the required Python packages using the following command:

                  pip install -r requirements.txt

### Dependencies
* Python 3+

      Python Packages
      * Flask
      * Wikipedia-API
      * wolframalpha
      * Kivy
      * Pypercard
